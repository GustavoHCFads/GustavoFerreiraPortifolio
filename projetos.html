Plano de estudos personalizado

1. Fundamentos da Web (Semanas 1 a 2)

Objetivo: Entender os conceitos básicos de HTML, CSS e JavaScript.

  • Semana 1
    • HTML5: Estrutura de páginas web, elementos semânticos, formulários, links, imagens.
    • CSS3: Estilização básica, layout, Flexbox, Grid, media queries.
  • Semana 2
    • avaScript: Tipos de dados, variáveis, funções, loops, condicionais, DOM manipulation.
    • Prática: Desenvolver uma landing page interativa com formulário de contato.

2. Versionamento de Código e Colaboração (Semana 3)

Objetivo: Aprender controle de versão com Git e colaborar com outros desenvolvedores.

  • Git: Repositórios, commits, branches, merge.
  • GitHub/GitLab: Como criar um repositório remoto, forks, pull requests.
  • Prática: Criar um repositório no GitHub e trabalhar em um projeto colaborativo.

3. Fundamentos da Web (Semanas 4 a 6)

Objetivo: Desenvolver interfaces dinâmicas usando bibliotecas/frameworks modernos.

  • Semana 4
    • JavaScript avançado: Arrays, objetos, funções de alta ordem, async/await, promises.
  • Semana 5
    • React.js: Componentes, estado, props, eventos.
    • Prática: Criar uma aplicação simples em React (ex: To-do list).
  • Semana 6
    • React avançado: Hooks (useState, useEffect), roteamento com React Router.
    • Gerenciamento de estado global com Context API ou Redux.

4. Backend com Node.js (Semanas 7 a 9)

Objetivo: Aprender a criar APIs e conectar com o frontend.

  • Semana 7
    • Node.js e NPM: Configuração, módulos, package.json.
    • Express.js: Criar uma API REST simples.
  • Semana 8
    • Banco de dados: SQL vs NoSQL.
    • MongoDB: Conectar ao Node.js usando Mongoose, operações CRUD.
  • Semana 9
    • Autenticação: JWT, autenticação via token.
    • Middleware no Express.js.
